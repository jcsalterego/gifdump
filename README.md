gifdump
=======

Dumps info about GIFs.

### Build

    $ make all

### Usage

    $ gifdump some.gif
    valid:	      yes
    width:	      430
    height:	      289
    global color table
           present:	1 (yes)
           color res: 1 (2 bits/pixel)
           sort flag: 1 (desc)
           size:      6 (128)
           bgcolor index:	0
    pixel aspect ratio:	0
